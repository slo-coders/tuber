# Tuber

# Create a database
After installing a PostgreSQL server, create a new databases called *tuber*. 

```bash
createdb tuber

# On Windows run the executable
# createdb.exe tuber
```
>NOTE:
>If testing, create a second database for testing called *tuber-test*.

# 'npm i'
# 'npm run seed'
# 'npm run start:dev'
